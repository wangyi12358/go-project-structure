# Go Project Skill

This skill provides comprehensive guidance for Go project development.

## What's Included

- **Project Structure**: Templates for monolithic and microservice architectures
- **Common Commands**: Frequently used Go commands
- **Code Templates**: Ready-to-use code patterns
- **Best Practices**: Industry-standard recommendations
- **Naming Conventions**: Package, file, and interface naming guidelines
- **Configuration Templates**: Makefile, Dockerfile, docker-compose examples

## Usage

This skill is automatically loaded when working with Go projects. It provides:

1. Project structure recommendations
2. Code generation templates
3. Common command references
4. Best practice guidelines

## Files

- `SKILL.md` - Main skill documentation

## Quick Reference

```bash
# Initialize Go module
go mod init github.com/username/project

# Build application
go build ./cmd/app

# Run tests
go test ./...

# Format code
gofmt -w .

# Run linter
golangci-lint run
```
